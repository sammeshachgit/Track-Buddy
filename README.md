# Track Buddy

Track Buddy is a modern fitness motivation app built with React Native and Expo. It helps users stay consistent with their workout routine through daily check-ins, streak tracking, photo-based progress logs, and group accountability.

## ✨ Why Track Buddy?

Track Buddy makes fitness feel social, visual, and rewarding. Instead of just logging workouts, users can:
- build a daily streak
- record progress with photos
- review their workout history
- create or join fitness squads
- stay motivated through a shared activity feed

## 🚀 Features

- Daily workout check-in flow
- Streak tracking for consistency
- Photo upload support for accountability
- Monthly progress calendar
- Personal profile dashboard with workout stats
- Squad/group creation and social motivation
- Clean, dark-themed mobile UI

## 🛠️ Tech Stack

- React Native
- Expo
- React Navigation
- AsyncStorage for local persistence
- Expo Image Picker
- Date-fns

## ▶️ Getting Started

1. Clone the repository
   ```bash
   git clone <your-repo-url>
   cd <your-project-folder>
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the app
   ```bash
   npm start
   ```

4. Open the project in Expo Go on your phone or run it in an emulator/simulator.

## 📌 Notes

- The current version uses local storage for users, attendance records, and groups.
- This is an MVP/prototype and can be expanded with Firebase, cloud sync, notifications, and social features in future updates.
